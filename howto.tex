\chapter{Prompt Context Snippets}
\label{chap:appendix-context-snippets}

This chapter contains the context snippets that were used to prompt the language model for generating the code translations. The snippets are organized by the type of context they provide, such as sandbox configuration files, build tool configurations, schema definitions, query examples, and other relevant code fragments.
